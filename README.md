# dgps_ros

#### 介绍
差分GPS模块的ROS包


#### 安装教程

1.  mkdir -p catkin/src
2.  cd catkin
3.  catkin_make
4.  cd src/
5.  wget https://gitee.com/rivercloud/dgps_ros/repository/archive/master.zip
6.  unzip master
7.  cd ..
8.  catkin_make

#### 使用说明

1. roslaunch dgps_ros dgps.launch
